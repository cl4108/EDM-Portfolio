Visualizing the network¶

We have tracked each classmates responses and posts within the class discussion to see who commented to whom. This discussion board does not require the original poster to reply to the commenters, so it is a one way relationship, and we may see one poster may get the most comments from different classmates.

density = nx.density(G)

                                             ### See tutorial here: https://www.geeksforgeeks.org/python-output-formatting/

The edge density is: 0.143

This value of 0.143 indicates that this network is not well connected because we can see less than 20% links among all possible links. This is probably because the class does not require orginal poster to reply to the commenters. This is a one way commenting system, but we can see there are people who has most links, may have a greater centrality within this network than others. Examples as Ty Scaletta and Xiaoyue Zhang
